# blockr.dplyr: Interactive 'dplyr' Data Transformation Blocks

Extends 'blockr.core' with interactive blocks for visual data wrangling
using 'dplyr' and 'tidyr' operations. Users can build data
transformation pipelines through a graphical interface without writing
code directly. Includes blocks for filtering, selecting, mutating,
summarizing, joining, and arranging data, with support for complex
expressions, grouping operations, and real-time validation.
